# FXML Resources

This directory contains JavaFX FXML layout files:

## Main Views
- `main-view.fxml` - Main application window
- `customers-view.fxml` - Customers list view
- `service-types-view.fxml` - Service types list view
- `orders-view.fxml` - Work orders list view
- `backup-view.fxml` - Backup management view
- `settings-view.fxml` - Settings view

## Dialogs
- `customer-edit-dialog.fxml` - Customer add/edit dialog
- `service-type-edit-dialog.fxml` - Service type add/edit dialog
- `new-order-dialog.fxml` - New work order creation dialog
- `order-details-view.fxml` - Order details view

FXML files define the UI layout and are loaded by controllers.
