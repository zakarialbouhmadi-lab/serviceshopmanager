# UI Controllers Package

This package contains JavaFX controllers for the user interface:

## Main Controllers
- `MainController.java` - Main window controller
- `CustomersController.java` - Customer management view
- `ServiceTypesController.java` - Service types management view
- `OrdersController.java` - Work orders list view
- `BackupController.java` - Backup management view
- `SettingsController.java` - Application settings view

## Dialog Controllers
- `CustomerEditDialogController.java` - Add/Edit customer dialog
- `ServiceTypeEditDialogController.java` - Add/Edit service type dialog
- `NewOrderDialogController.java` - Create new work order dialog
- `OrderDetailsController.java` - View work order details

Controllers handle user interactions and connect UI with services.
