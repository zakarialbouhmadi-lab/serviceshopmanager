# ViewModels Package

This package contains view model classes for JavaFX data binding:

- `CustomerViewModel.java` - Observable wrapper for Customer
- `WorkOrderViewModel.java` - Observable wrapper for WorkOrder
- `WorkOrderItemViewModel.java` - Observable wrapper for WorkOrderItem
- `ServiceTypeViewModel.java` - Observable wrapper for ServiceType

ViewModels use JavaFX properties (StringProperty, ObjectProperty, etc.)
for automatic UI updates when data changes.
